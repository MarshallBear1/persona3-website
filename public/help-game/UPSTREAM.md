# Upstream game foundation

This embedded game is adapted from [JSLegendDev/Pokemon-p5js](https://github.com/JSLegendDev/Pokemon-p5js).

- Upstream author: JSLegend
- Upstream license: MIT
- Reused foundation: p5.js scene loop, animated player controller, tiled map renderer, camera, collision handling, NPC interaction, dialog box, and supporting game assets
- Local adaptation: WASD controls, direct world entry, support-ticket interaction event, and host-page integration

The original MIT license is preserved in `LICENSE`.
